edit_note Quill WYSIWYG Редактор (ngx-quill) Використання Delta (JSON) формату для збереження контенту
Тільки читання info Інформація про формати

✅ Переваги Delta (JSON) формату:

  • Точне збереження структури - без втрати форматування
  • Незалежність від HTML - міграція між редакторами
  • Версіювання - зручне відстеження змін
  • Безпека - немає проблем з XSS атаками
  • Продуктивність - швидкий парсинг та рендеринг

⚠️ Проблеми HTML формату:

  • Втрата форматування при парсингу
  • Різні браузери рендерять по-різному
  • Проблеми з безпекою (XSS)
  • Складність міграції на інші редактори
  • Quill v2 має issues з getSemanticHTML()

🔧 Налаштування компонента:

  • format="object" - використання Delta (JSON)
  • sanitize="true" - санітизація контенту
  • debounceTime="300" - затримка оновлення моделі
  • readOnly - режим тільки для читання
  • modules - конфігурація toolbar та модулів

📋 Приклад Delta структури:

{{'{
  "ops": [
    { "insert": "Hello " },
    { "insert": "World", "attributes": { "bold": true } },
    { "insert": "!\\n" }
  ]
}'}}
