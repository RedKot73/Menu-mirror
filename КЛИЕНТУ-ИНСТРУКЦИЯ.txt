ШАГ 1: РАСПАКОВКА

Распакуйте архив S5Server-2025-11-11.zip в любую директорию, например:
  C:\S5Server\

Откройте браузер и перейдите по адресу:
  http://localhost:5000

ИЗМЕНЕНИЕ ПОРТА (если 5000 занят)
================================================================================

1. Откройте файл: appsettings.Production.json
2. Найдите строку: "Url": "http://0.0.0.0:5000"
3. Измените порт на свободный, например: "Url": "http://0.0.0.0:8080"
4. Сохраните файл
5. Запустите start.bat

================================================================================
РЕЗЕРВНОЕ КОПИРОВАНИЕ
================================================================================

ВАЖНО! Регулярно создавайте резервные копии базы данных:

Файл для backup: Data\S5_DB.sqlite

Как сделать backup:
1. Остановите приложение (закройте окно или Ctrl+C)
2. Скопируйте файл Data\S5_DB.sqlite в безопасное место
3. Запустите приложение снова

Рекомендуется делать backup:
- Ежедневно
- Перед обновлением системы
- После важных изменений данных

================================================================================
ОБНОВЛЕНИЕ СИСТЕМЫ
================================================================================

Когда получите новую версию:

1. Остановите приложение
2. ОБЯЗАТЕЛЬНО сделайте backup: Data\S5_DB.sqlite
3. Замените все файлы КРОМЕ:
   - Data\S5_DB.sqlite (ваша база данных)
   - appsettings.Production.json (ваши настройки)
4. Запустите start.bat

================================================================================
УСТРАНЕНИЕ НЕПОЛАДОК
================================================================================

ПРОБЛЕМА: "Порт 5000 уже используется"
РЕШЕНИЕ: Измените порт в appsettings.Production.json (см. выше)

ПРОБЛЕМА: "Не удается запустить приложение"
РЕШЕНИЕ: 
  - Запустите start.bat от имени администратора
  - Проверьте брандмауэр Windows (добавьте S5Server.exe в исключения)
  - Проверьте антивирус (добавьте папку в исключения)

ПРОБЛЕМА: "База данных недоступна"
РЕШЕНИЕ:
  - Проверьте наличие файла Data\S5_DB.sqlite
  - Проверьте права доступа к файлу
  - Восстановите из backup

ПРОБЛЕМА: "Белый экран в браузере"
РЕШЕНИЕ:
  - Проверьте наличие папки wwwroot
  - Очистите кеш браузера (Ctrl+Shift+Delete)
  - Попробуйте другой браузер
  - Нажмите Ctrl+F5 для жесткого обновления

ПРОБЛЕМА: "Медленная работа"
РЕШЕНИЕ:
  - Перезапустите приложение
  - Проверьте размер базы данных (Data\S5_DB.sqlite)
  - Закройте другие приложения

РЕКОМЕНДАЦИИ:

1. Используйте сложные пароли (если добавлена аутентификация)

ОГРАНИЧЕНИЕ ДОСТУПА:

По умолчанию приложение доступно только на локальном компьютере.
Для доступа из сети измените в appsettings.Production.json:
  "Url": "http://0.0.0.0:5000"  - доступ из сети
  "Url": "http://127.0.0.1:5000" - только локальный доступ
